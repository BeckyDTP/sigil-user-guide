Manage Plugins

— Extend The Functionality Of Sigil —

You can extend the functionality of Sigil by installing Python 3 plugins. (Since Sigil comes with an embedded Python 3 interpreter and several Python libraries optimized for Sigil, you won’t have to install Python 3 on your system.)

Install Plugins:

To install a plugin:

  1. Download the desired plugin .zip file.
    (You can find an up-to-date list of all supported Sigil plugins on MR.)
  2. Select Plugins > Manage Plugins and click Add Plugin.
  3. Select the plugin .zip file and click OK.
  4. Enable Use Bundled Python, if it isn’t already enabled.
    (If Use Bundled Python can’t be selected, click Auto or Set to select an external Python interpreter.)

Don’t unpack or rename plugin .zip files!

To run a plugin simply select it from the Plugins menu. For example, to run the FlightCrew plugin select Plugins > Validation > FlightCrew.

Assign Plugins To Toolbar Buttons:

You can assign the ten most frequently used plugins to Plugin toolbar buttons. (To assign a plugin to a Plugin toolbar button select it from one of the five Assign as Plugin list boxes.)

Troubleshooting:

Error: Interpreter python3.4 has no path set

This error message is usually displayed if Use Bundled Python is disabled and the ≥Python 3.4 text box is empty or contains an invalid path. Click Auto or Set to select an external Python interpreter or enable Use Bundled Python.

Error: Plugin is not a valid Sigil plugin

This error message is usually displayed if the plugin .zip file is invalid or the plugin .zip file was renamed. If you renamed the plugin .zip file, re-download it and retry the installation.

If you’re still experiencing problems, install and run the Sigil test plugin and post the results in the MR Sigil Plugins forum.

Write Your Own Plugins:

Thanks to a comprehensive Python framework it is relatively easy to write plugins for Sigil. If you’re interested in writing your own Python 3 Sigil plugins, have a look at the Sigil Plugin Framework guide.
