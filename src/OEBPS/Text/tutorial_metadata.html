Add the Author and Title

In addition to the actual content of your book, EPUB files also contain book details like Author and Title. These can be added with Sigil.

To add details (called metadata) about your book to your EPUB, click the Metadata Editor button or the menu item Tools→Metadata Editor:

The Metadata Editor window will be displayed:

The entries required by an EPUB are Title, Author, and Language.

To add the Title and Author to your book:

  • Title: Click dc:title > [no data] and type the full title of your book, for example enter Alice in Wonderland.
  • Author: Click Add Metadata > Author and type the author’s name, for example enter Lewis Carroll.
  • File-As: This is actually optional. But it is usually included as it controls how the author’s name is sorted. Click dc:creator > Add Property > File as and type the author’s name as it should be sorted, for example enter Carroll, Lewis.

Once you have updated the information, you can click OK and then save your EPUB again.

You may have noticed that there is a lot of room for other values in the Metadata Editor. Only the Title, Author, and Language are required for EPUBs, but if you want to add more values using the Add Metadata or Add Property buttons you can.
