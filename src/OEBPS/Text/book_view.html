Book View

— WYSIWYG Editing —

Book View (or WYSIWYG mode – what you see is what you get) was removed from Sigil in Version 0.9.15. If you need the ability to edit in a WYSIWYG environment, the maintainers of Sigil have created a standalone app called PageEdit that can be used via Sigil's Open With feature. It can also be defined as the default external html editor and assigned a shortcut
