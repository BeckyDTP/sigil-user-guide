Special External Xhtml Editor

— Using Something like PageEdit Edit Xhtml Files —

Users can use something like the PageEdit program to edit an epub's xhtml files.

By setting the path to the external editor's executable in Sigil's preferences (Edit->Preferences->General Settings->Set your preferred alternative external xhtml editor), they can then launch the Book-View-selected file using the Launch External Xhtml Editor button on the Sigil toolbar (the icon looks like a paper and pencil). They can also use/assign the keyboard shortcut for the button to launch the editor (defaults to F2).
